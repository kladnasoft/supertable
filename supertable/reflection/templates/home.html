Your powerful data management solution with Super Tables, SQL execution, monitoring, and AI-powered analytics - all in one intuitive interface.

Quick Start Guide

1

Create a Super Table

Start by creating a Super Table to organize your related data tables. Click the "Super Tables" menu and use the creation controls.

2

Key Features

Super Tables

Manage your data effortlessly with Super Tables — smart collections of related tables designed to streamline organization and access control. Unlike traditional databases, Super Tables let you query across multiple tables directly, without the need for unions.

SQL Execution

Write and execute queries effortlessly with our advanced SQL editor — featuring syntax highlighting, autocomplete, and real-time, paginated results. Simply click on a Super Table to expand its contents and run your queries instantly.

Data Import Hub

Quickly upload CSV, JSON, and PARQUET files with built-in preview and automatic schema detection. Use the staging area to validate your data before importing. You can create a new table on your first upload or store files for Data Staging, allowing you to reload them anytime as needed.

AI Tools

Unlock AI-powered tools for query generation, data analysis, and natural language to SQL conversion. With Docu Genie, a semantic layer is automatically created, giving you full visibility into your available data.

Query Monitoring

Monitor query performance with detailed statistics, visualized execution plans, and comprehensive write operation metrics — giving you deeper insights into your data flow.

User Management

Administer user roles and permissions with fine-grained access control for your organization's data.
